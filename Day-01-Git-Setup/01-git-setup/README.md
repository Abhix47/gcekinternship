# Day 1 - Git Setup

## Objective
Learn the basics of Git and GitHub version control.

## Activities Performed
- Installed Git
- Configured Git username and email
- Created a local repository
- Learned basic Git commands
- Connected local repository to GitHub
- Performed commit and push operations

## Commands Practiced

```bash
git init
git add .
git commit -m "First commit"
git remote add origin <repository-url>
git push -u origin main
```

## Outcome
Successfully set up Git and GitHub and uploaded files to a remote repository.
