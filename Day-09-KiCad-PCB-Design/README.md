# Day 9 - KiCad PCB Design

## Objective

Explore the PCB design workflow using KiCad and understand the process of converting electronic schematics into PCB layouts.

## Topics Covered

- Introduction to KiCad
- Creating electronic schematics
- PCB layout design
- Component placement
- Routing electrical connections
- PCB design workflow
- Design Rule Check (DRC)

## Software Used

- KiCad

## Learning Outcome

This session introduced the fundamentals of PCB design using KiCad. It focused on creating PCB layouts, understanding component placement, and learning the overall PCB design workflow for embedded systems and IoT applications.
