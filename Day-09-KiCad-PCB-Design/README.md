# Day 9 - KiCad PCB Design

## Objective

Understand the PCB design workflow using KiCad, including schematic creation, component placement, routing, and PCB layout design.

## Topics Covered

- Introduction to KiCad
- Creating electronic schematics
- PCB layout design
- Component placement
- Routing electrical connections
- PCB design workflow
- Design Rule Check (DRC)

## Software Used

- KiCad

## Learning Outcome

Developed an understanding of the PCB design process using KiCad, including schematic creation, PCB layout design, component placement, routing, and Design Rule Checking (DRC). These concepts provide the foundation for designing printed circuit boards for embedded systems and IoT applications.
