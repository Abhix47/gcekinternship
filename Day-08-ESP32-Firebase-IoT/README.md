# Firebase Web Control

## Objective

Create a web application hosted on Firebase to remotely control an ESP32 LED.

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Hosting
- Firebase Realtime Database

## Features

- LED ON/OFF control
- Realtime Database integration
- Hosted using Firebase Hosting

## Live Website

https://ledcontrolproject-a827f.web.app/

## Result

Successfully controlled an ESP32 LED remotely through a Firebase-hosted web interface.
