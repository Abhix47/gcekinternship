# Day 4 - Software Setup

## Objective
Set up the software tools required for the internship.

## Software Installed

### KiCad
- Open-source PCB design software
- Used for schematic capture and PCB layout

### Eclipse Temurin JDK 25
- Java Development Kit
- Required for Java-based tools and development

### CP210x USB Driver
- USB-to-UART driver for ESP32 communication
- Enables the computer to detect and communicate with the ESP32 board

### VS Code
- Code editor used for development

### PlatformIO
- Embedded systems development platform
- Used for programming ESP32 boards

## Outcome
Successfully installed and verified all required software and drivers for future ESP32 and hardware-based sessions.
